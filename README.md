# Fable-5-Experiments
How far can Fable 5 go!

## Arcade Soccer

A 4v4 top-down arcade soccer game — vanilla JS + Canvas, no dependencies.

**Play it:** https://iamsorenl.github.io/Fable-5-Experiments/

Single player vs AI (Easy/Normal/Hard) or local 2-player on one keyboard.
